# Car-Rental-Management-Project
- Management of an Car Rental Business
This is a simple project that i have done for my .Net branch in master degree
There are four modules: Cars, Users, Reservationsa and Booking.
In this application you can manage users that can be user or admin depend on funcionalities.
Also you have to CRUD cars and than the Salesman has the opportunity to give the cars for renatal with a user friendly features.
